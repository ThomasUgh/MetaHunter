# MetaHunter

A metadata extraction tool for images and PDFs.

## Features

- Extract EXIF data from images (JPEG, PNG, TIFF, WebP)
- Extract PDF metadata (author, dates, page count)
- Image preview
- Drag & drop support
- Clean dark interface

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- Tailwind CSS
- exifr, pdf-lib
