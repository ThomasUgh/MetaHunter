# MetaHunter

A metadata extraction tool for multiple file types.

## Features

- Images: EXIF, GPS, camera info
- PDF: author, dates, page count
- Office: DOCX, XLSX, PPTX metadata
- Audio: ID3 tags, album art
- Video: container metadata

## Getting Started

```bash
npm install
npm run dev
```
